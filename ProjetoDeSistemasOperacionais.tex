\def \objetivos	{Os principais objetivos da disciplina são fornecer aos alunos os fundamentos e os detalhes do projeto de um sistema operacional. Ao final da disciplina, espera-se que os alunos sejam capazes de: entender a arquitetura e o funcionamento geral dos principais componentes de um sistema operacional; descrever os problemas e as respectivas soluções teóricas que são normalmente encontrados no projeto de um sistema operacional; projetar e implementar soluções para problemas de programação concorrente/paralela utilizando threads e processos.
}

Sistemas de Arquivo: organização; métodos de acesso. Sistemas de E/S.
